Overview

The Game is a third-person Unity 3D action project built as my first larger game development exercise.
The project showcases game mechanics, AI behavior, NavMesh pathfinding, level design, particle effects, physics interactions and UI systems implemented in C#.

The game contains:

- Unity 3D environment
- C# gameplay scripts
- NavMesh pathfinding
- Enemy AI behavior
- Laser and particle effects
- Physics-based interactions
- Custom UI and menus
- Fully playable Windows build (available in Releases)

Features

- Third-person player controller
- Enemy AI with detection & chasing
- NavMesh-based navigation
- Projectile & laser weapon system
- Level obstacles & platform structures
- Custom materials & sound effects
- Basic health and hit reaction logic
- Simple HUD (health, crosshair, etc.)

Hardware Requirements
Minimum Requirements

These are the lowest specs at which the game is expected to run:

- CPU	Intel i5 (4th gen) / AMD equivalent
- GPU	NVIDIA GTX 660 / AMD Radeon HD 7870
- RAM	8 GB
- Storage:	1 GB free space
- OS:	Windows 10 / 11 (64-bit)
  
Recommended Requirements
For smooth performance (60 FPS):

- CPU	Intel i5 (8th gen) / AMD Ryzen 5
- GPU	NVIDIA GTX 1060 / AMD RX 580
- RAM	16 GB
- Storage:	SSD recommended
- OS: Windows 10 / 11 (64-bit)

Note: The game is not optimized for low-end hardware. Integrated GPUs may struggle to maintain stable performance.

Download & Installation

1. Go to Releases
2. Download: The_Game_exe.zip
3. Extract the ZIP file
4. Run The Game.exe

No installation required.
